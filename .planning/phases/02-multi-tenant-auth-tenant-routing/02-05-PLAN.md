      - `memberships.length === 1` -> `redirect(\`/shop/${memberships[0].tenant.slug}/dashboard\`)`
      - `memberships.length > 1` -> render a lightweight tenant picker (a simple list of `<Link href={\`/shop/${m.tenant.slug}/dashboard\`}>{m.tenant.name}</Link>`; Claude's discretion per D-09, a plain list is fine — only Roy/dev will have >1 in v1).

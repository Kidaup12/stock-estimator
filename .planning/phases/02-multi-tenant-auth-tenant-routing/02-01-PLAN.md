    Create `lib/tenant/slug.ts`. No npm dependency (research: a dep is overkill). Implement:
    ```typescript
    /** Slugify a shop name into a URL segment: lowercase, alphanumerics joined by hyphens, no leading/trailing hyphen. */
    export function slugify(name: string): string {
      return name
        .toLowerCase()
        .normalize("NFKD")
        .replace(/[^a-z0-9]+/g, "-")
        .replace(/^-+|-+$/g, "");
    }
    ```
    This is consumed by the onboarding flow (Plan 05) which slugifies the shop name on tenant creation (D-07). Export it as a named export so Plan 05 and the backfill script import `{ slugify }`.
